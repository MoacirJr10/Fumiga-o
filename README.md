## ⚙️ Funcionalidades

<!-- EXEMPLO DE FUNCIONALIDADES: -->
- [x] Cálculo de Metragem Cúbica:
O usuário insere as dimensões (altura, largura e comprimento) de um espaço em metros.
O sistema calcula a metragem cúbica com base nos valores fornecidos.
Exemplo de uso: calcular o volume de um armazém ou contêiner para realizar fumigação.

- [x] Cálculo de Quantidade de Produtos de Fumigação:
Com a metragem cúbica calculada, o sistema calcula automaticamente as quantidades de diferentes tipos de produtos de fumigação, como:
Sachê: Dividido pela metragem cúbica.
Pastilha: Calculado multiplicando por um valor fixo.
Comprimido: Multiplicado por um valor fixo.
Exemplo de uso: determinar a quantidade correta de produtos de fumigação para um espaço específico.

## 🛠 Tecnologias

HTML (HyperText Markup Language):
Utilizado para estruturar o conteúdo da aplicação, incluindo a criação de elementos como entradas de texto, botões e divisões.

CSS (Cascading Style Sheets):
Utilizado para estilizar a aplicação, definindo cores, fontes, espaçamentos e layouts para tornar a interface do usuário visualmente atraente.

JavaScript:
Utilizado para implementar a lógica da aplicação, permitindo que o usuário interaja com a interface (por exemplo, ao clicar no botão para calcular a metragem cúbica e exibir os resultados).
Frameworks e bibliotecas (opcional):

Se você decidir usar, pode integrar frameworks como Bootstrap ou bibliotecas JavaScript como jQuery para melhorar a interface e a funcionalidade.
